# BPA 시스템 매뉴얼
## 1. 개요
* 메인 URL : https://jwchob-lab.github.io/bpa/
* 사용 요건 : HTML5 지원 최신 버전 브라우저 (크롬, 엣지 등)
* 기술 스택 : 기능에 따라 PyScript, JavaScript, AI 등
* AI 활용 : 구글 제미나이 API 무료키 사용 (사용시 모델 학습 활용 이슈)
* 저장 서버 (GitHub) : 리포지토리 용량 1GB, 빌드 시간당 10회, 월간 대역폭 100GB
## 2. 구조
### 3.1. 프레임 구조
* **영역 1: Sidebar (L)**
    - 서비스 로고 및 버전 정보
    - `links.js` 로딩 상태 모니터링 및 리로드 기능
    - 앱 카테고리별 내비게이션 메뉴 자동 생성
* **영역 2: Topbar (TR)**
    - 선택된 앱의 타이틀 및 소스 경로 출력
    - 외부 브라우저 실행을 위한 '새탭' 바로가기 지원
* **영역 3: Viewer (BR)**
    - 실제 기능이 작동하는 `iframe` 영역
    - 초기 상태: ASCII Art 기반의 시스템 터미널 로딩 화면 출력
### 3.2. 계층 및 경로
* 시스템의 유지보수 효율을 위해 레이아웃, 디자인, 콘텐츠 영역을 분리하여 관리합니다.
```text
./bpa (레파지토리 루트)
└── index.html                     # 메인 엔트리 포인트 (사이드바 및 프레임워크 정의)
└── theme.css                      # 전역 디자인 시스템 (컬러 토큰 및 공통 스타일)
└── readme.html                    # 시스템 도움말 (원본: readme.md)
└── links.js                       # app 호출 링크파일
└── app/<$appname>                 # 프로그램 실행 파일 (../theme.css 참조)
└── .github/workflows/deploy.yml   # 깃허브 자동 배포(CI/CD) 사용시 필요
└── .git/                          # 프로그램 구동 무관 (로컬 개발환경과 깃허브 연동)
```
