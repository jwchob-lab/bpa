개요

  • 별도 설치 없이 웹 브라우저에서 실행 가능
  • 깃허브에 각 프로그램 레파지토리 생성 후 배포
  • HTML 파일 기준으로 생성되며 스크립트는 기능에 따라 상이함 (PyScript, JS, AI 등)
  • 링크 관리는 links.json 파일로 별도 관리
  • DB 필요 시 깃허브와 연동된 Supabase DB 호출

업무자동화 (BPA)

BPA 메인 시스템
  • URL : https://jwchob-lab.github.io/bpa/

일반 운영 항목

트리거 파악
  • 배포 전 트리거 파악 프로그램 배포 및 내부 공유 (/w 공통 DBA 4인)
  • 배포파일(1번) 및 트리거 사용 테이블 목록 파일(2번)을 창에 드래그하여 실행
디스크 관리
  • 디스크 Full 예상 그래프 생성 (월 단위)
  • 표 형식 TXT 파일 붙여넣기 또는 CSV 파일 업로드 지원
  • 현재 월까지의 디스크 증가량 입력
디스크 관리 (AI)
  • 제미나이 API 사용하여 예측 그래프 생성
  • 이미지 파일 캡처하여 붙여넣기 활용 가능 (GB 기준)
  • → [이슈] 이미지 인식 시 실제값보다 낮은 값으로 인식되는 현상 존재
보고서 작성 (AI)
  • 제미나이 API를 통한 텍스트 자동 보정 및 최적화

데이터베이스 관리

오라클 권한백업
  • 테이블 권한 백업 요청 시 테이블 목록 입력 후 자동 실행
오라클 XPLAN
  • 유저명 및 튜닝용 쿼리 입력을 통한 XPLAN 생성문 자동 생성
DB 자유게시판
  • 깃허브에서 Supabase DB 호출 연동
  • PostgreSQL 17.6 (aarch64-linux) 환경
  • 실제 운영 시 보안 파일 분리 및 DB 권한 보완 필요
